# Gutenberg subset - terms search and similar documents #


### ¿Para que es este repositorio? ###

En este repositorio se encuentran 3 carpetas para llevar a cabo el desarrollo de un proyecto para generar un indice invertido a partir de un
dataset sacado de la colección de gutenberg, tambien hay una funcion para obtener los documentos mas similares a los demas y por ultimo una aplicaion web para interacutar con estos procesamientos, buscando documentos por palabras y viendo en un listado los 10 documentos mas similares a algun documento seleccionado.

#### Instrucciones
Cada carpeta contiene las instrucciones para ejecutar el correspondiente proceso. El orden en que se deben seguir las instrucciones por carpeta es:
	- inverted_index
	- cosine_similarity
	- gutenberg_webapp

En cada carpeta se explica que es y como se ejecuta cada proceso y actividad.
